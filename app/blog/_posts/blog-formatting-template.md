## Section Heading (H2)

Use **bold text** to highlight key points.
Use *italic text* for emphasis.
You can combine both as ***bold italic*** when needed.

### Subheading (H3)

Add internal links like [Contact Us](/contact) and [Our Services](/services).
Add external links like [Google](https://www.google.com).

## Paragraph Spacing Tips

Keep one blank line between paragraphs.

Use short paragraphs for readability on mobile screens.

Avoid very long text blocks unless needed.
